## Baseline methodology (honest)

The `baseline_protocol_sift` block in ground truth JSON encodes **estimated** prompt-only recall (0.65 ghost, 0.50 orphan) from typical agent behavior without an architectural verifier. GRAVEYARD does not claim live head-to-head runs against Council-SIFT or EvidenceChain — we compare against documented baseline estimates and ship reproducible JSON.

Simulated baseline **cannot** catch hallucinations (catch rate 0.0). GRAVEYARD verifier catches 100% of injected tests on the sample case.
